# IP_Smart_LMS – IP 기반 스마트 출석 시스템

> **교내 IP 기반 부정 출석 방지용 웹 출석 시스템**  
> FastAPI + MongoDB + Render 로 구현한 간단한 LMS 출석 데모입니다.

---

## 👀 프로젝트 소개

IP_Smart_LMS는 **수업 시간에 학생들이 입력하는 출석 코드와 접속 IP**를 함께 기록하여  
교내 WiFi / 내부망 / 외부망 등을 구분하고, **의심스러운 출석을 한눈에 확인**할 수 있도록 만든 웹 서비스입니다.

- 교수님은 수업 날짜별로 **6자리 출석 코드를 생성**해서 학생들에게 공유합니다.
- 학생은 본인의 이름과 출석 코드를 입력해 출석을 남깁니다.
- 시스템은 **접속 IP를 자동으로 기록하고, 교내/외부망 여부를 판정**합니다.
- 교수님은 코드별 상세 화면에서 **출석 현황과 의심 IP 목록**을 바로 확인할 수 있습니다.

---

## ✨ 주요 기능

### 1. 메인 랜딩 페이지 (`/`)
- “학생용 출석” / “교수용 출석 관리” 버튼 제공  
- 사용자는 역할에 맞는 페이지로 이동

### 2. 학생용 출석 페이지 (`/student`)
- 이름 입력
- **수업 선택 드롭다운**
  - 현재 유효한 출석 코드가 있는 수업만 표시
  - 예: `2025-12-01 (유효 종료: 11/30 19:13)`
- 6자리 출석 코드 입력
- 출석 결과 메시지 표시
  - `출석이 정상적으로 처리되었습니다.`
  - 이미 출석한 경우: `이미 출석이 처리되었습니다.` + 버튼 비활성화
- 접속 IP에 따른 안내 문구
  - ✅ 교내 WiFi
  - ⚠ 내부망/인접 강의실
  - ❗ 외부망(LTE 등)

### 3. 교수용 출석 관리 페이지 (`/teacher`)
- **수업 날짜 + 유효 시간(분)** 을 입력하여 6자리 출석 코드 생성
- 현재 유효한 출석 코드 목록
  - 수업 날짜, 출석 코드, 유효 종료 시각, 남은 시간(진행중/만료됨)
  - 행 클릭 시 해당 코드의 상세 화면으로 이동
- 지난 출석 목록
  - 만료된 출석 코드들의 기록

### 4. 출석 코드 상세 화면 (`/teacher/code/{code_id}`)
- 코드 정보
  - 출석 코드, 수업 날짜, 상태(진행 중 / 만료), 남은 시간
- 출석 현황
  - 학생 이름, IP, IP 상태, 출석 시간
- IP 주소 확인 필요 목록
  - `NORMAL` 이 아닌 IP만 따로 표시 (DEV/WARNING/SUSPICIOUS)
